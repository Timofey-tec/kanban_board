# Kanban Board

Полнофункциональная Kanban-доска с drag-and-drop — портфолио-проект, демонстрирующий управление сложным состоянием в React через `useReducer`, плавное DnD с `@dnd-kit` и проработанный UX.

![demo](./demo.gif)

## Функционал

### Колонки
- 3 колонки по умолчанию: «To Do», «In Progress», «Done»
- Добавление колонок с произвольным названием
- Переименование колонки по клику на заголовок (inline-редактирование)
- Удаление колонки с подтверждением (при наличии карточек предупреждает об удалении)
- Перетаскивание колонок для изменения их порядка

### Карточки
- Добавление карточки через модальное окно (заголовок обязателен, описание — опционально)
- Редактирование карточки по клику — открывает модалку с предзаполненными полями
- Удаление с диалогом подтверждения — случайно не потерять
- Поля: заголовок (до 120 символов), описание, дата создания, цветной тег
- Теги: Bug (красный), Feature (синий), Urgent (оранжевый), Design (фиолетовый)
- Счётчик карточек в заголовке каждой колонки

### Drag-and-drop (`@dnd-kit`)
- Перетаскивание карточек между колонками
- Сортировка карточек внутри одной колонки
- Перетаскивание самих колонок
- Live-preview во время перетаскивания (DragOverlay)
- Курсор `grab` на карточках, видимый drag-handle при наведении

### Поиск и фильтрация
- Поиск по заголовку и описанию карточек в реальном времени
- Фильтрация по тегу (клик на Badge в тулбаре)

### Прочее
- Тёмная / светлая тема с переключателем, сохраняется между сессиями
- Полное сохранение состояния доски в `localStorage` при каждом изменении
- Кнопка «Reset Board» — возврат к демо-состоянию с подтверждением
- При первом запуске (и после сброса) показывает обучающие демо-карточки

## Стек технологий

| | |
|---|---|
| React 18 | UI и компонентная архитектура |
| TypeScript (strict) | `Card`, `Column`, `Tag`, `BoardState`, `BoardAction` — без `any` |
| @dnd-kit/core + sortable | Drag-and-drop с PointerSensor и DragOverlay |
| Vite | Сборка и dev-сервер |
| Tailwind CSS v4 | Стилизация, тёмная тема |

## Архитектура

```
src/
  components/
    Board/         – DnD-контекст, тулбар, оркестрация
    CardModal/     – Модалка создания/редактирования карточки
    shared/        – Button, ConfirmDialog
  hooks/
    useBoardState.ts    – useReducer + синхронизация с localStorage
    useLocalStorage.ts  – единственная точка доступа к localStorage
  types/
    board.ts       – Card, Column, Tag, BoardState, BoardAction
  utils/
    boardReducer.ts – чистые функции-редьюсеры (ADD_CARD, MOVE_CARD, …)
```

**State management:** `useReducer` с 11 типизированными экшенами. Одно действие = один экшен.  
**Persistence:** весь `localStorage`-доступ только через хук `useLocalStorage` — никогда напрямую в компонентах.
